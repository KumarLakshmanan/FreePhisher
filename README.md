# FreePhisher

## Description

> Phish Social medias by using FreeFire

```
[+] Facebook
[+] Instagram
[+] Twitter
[+] VK
[+] Google
```

## Usage

```
git clone https://github.com/KumarLakshmanan/FreePhisher
cd FreePhisher
chmod +x *
./FreePhisher.sh
```
